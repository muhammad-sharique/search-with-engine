Test the Search with Engine Extension

How to test:
  1. Load the extension in Chrome (chrome://extensions/)
  2. Select any text on this page
  3. Right-click on the selected text
  4. Click "Search with Google" (or your chosen engine)
  5. A new tab should open with search results

Paragraph to Test

Chrome extensions are small software programs that customize the browsing experience. They enable users to tailor Chrome functionality and behavior to individual needs or preferences. Extensions are built on web technologies such as HTML, JavaScript, and CSS.

Select any part of this text, right-click, and use the context menu to search. You can also test changing the search engine in the extension options and see how the context menu updates accordingly.

Configuration Testing

To test the options page:

  1. Right-click the extension icon in Chrome toolbar
  2. Select "Options"
  3. Change the search engine
  4. Return to this page and try searching again
  5. The context menu should now show your new search engine
